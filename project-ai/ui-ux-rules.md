# UI UX RULES

## DESIGN

Objetivo:

- moderno
- premium
- limpo
- sofisticado
- intuitivo

## INTERFACE

Garantir:

- consistência visual
- alinhamentos corretos
- espaçamentos equilibrados
- boa hierarquia visual

## EXPERIÊNCIA

Sempre utilizar:

- feedback visual
- loading
- animações suaves
- microinterações

## RESPONSIVIDADE

Garantir:

- mobile first
- adaptação total
- navegação intuitiva

## ACESSIBILIDADE

Garantir:

- contraste
- legibilidade
- acessibilidade
- navegação por teclado

## UX

Evitar:

- telas poluídas
- excesso de informação
- ações confusas
- fluxo complicado

## COMPONENTES

Padronizar:

- botões
- inputs
- modais
- cards
- tabelas
- menus

## SAAS PREMIUM

Visual inspirado em:

- Stripe
- Linear
- Notion
- Vercel
- Framer
- Airbnb

## PERFORMANCE

Garantir:

- carregamento rápido
- transições suaves
- experiência fluida
